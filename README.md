# Project about autonomous drone control with ROS2, PX4 and Gazebo.

This repository contains a series of labs focused on autonomous drone control using ROS2, PX4 and Gazebo. The labs cover various aspects of drone control, including basic movement, computer vision and swarm control.

## Tech stack

- ROS2 (Humble)
- PX4
- Gazebo
- OpenCV
- YOLOv8
